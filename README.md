# 2048 Game Collection

Коллекция различных версий игры 2048 на Python с единым стилем интерфейса.

## 🎮 Версии игр

### Основные версии:
- **Обычная** - классическая игра со случайным появлением двоек
- **Легкая** - двойки появляются в самых удобных для игрока местах
- **Сложная** - двойки появляются в самых неудобных местах

### Тренажеры:
- **Тренажер (медленный)** - с подсказками, подтверждением ходов и отменой
- **Тренажер (быстрый)** - автоматическое применение лучших ходов

## 🚀 Быстрый старт

### Запуск через главное меню (рекомендуется):
```bash
python main.py
```

### Прямой запуск конкретных версий:
```bash
# Обычная версия
python games/normal_game.py

# Легкая версия  
python games/easy_game.py

# Сложная версия
python games/hard_game.py

# Тренажеры
python solvers/practice_slow.py
python solvers/practice_fast.py
```

## 🎯 Управление

- **Кнопки управления** на интерфейсе или **WASD/Стрелки** для перемещения
- **Цель игры**: собрать плитку со значением 2048
- **Объединение**: одинаковые числа объединяются при столкновении

## 🎨 Особенности

- **Единый визуальный стиль** - все версии используют оригинальную цветовую схему 2048
- **Модульная архитектура** - легко расширять и модифицировать
- **Адаптивная сложность** - от обучения до настоящего вызова
- **Умные подсказки** - тренажеры помогают улучшить стратегию

## 🛠 Требования

- Python 3.6+
- Библиотека tkinter (обычно входит в стандартную поставку Python)

## 🔧 Разработка

Проект использует модульную структуру:
- `core/` - содержит игровую логику
- `ui/` - отвечает за отображение
- `solvers/` - алгоритмы подсказок и тренировки
- `hubs/` - меню и навигация
